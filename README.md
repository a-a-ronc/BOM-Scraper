# BOM-Scraper

Turn PDF drawings of conventional rack configurations into a structured **bill of materials** — with seismic-regulation logic built in.

Manual takeoff from rack drawings is slow and error-prone: reading elevations, counting uprights and beams, cross-checking seismic requirements, then retyping everything into an RFQ. This tool automates that cycle, eliminating hours of manual document preparation per quote.

## How it works

1. **Input** — PDF drawings of conventional rack configurations (selective rack elevations/plans)
2. **Extraction** — parses drawing content into component counts and specifications
3. **Seismic logic** — applies embedded seismic-regulation rules to the extracted configuration
4. **Output** — structured BOM ready for RFQ generation
